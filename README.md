# 新規事業立ち上げSLG（プロトタイプ）

起業家となって人材を集め、市場を分析し、製品を送り込んで事業を育てる——ターン制の経営シミュレーションゲームの**初期プロトタイプ**です。

## ▶ 遊ぶ（スマホ・PCどちらでも）

**プレイURL：** https://multi-line2025.github.io/newbiz-slg-prototype/

ブラウザで開くだけ。インストール・アカウント不要。スマホでも動きます。
（開くたび・「新規開始」のたびに違う市場・人材になります。）

## どんなゲーム？

- **人材**：世界中の候補者を**スカウト**で見抜いて採用（能力は隠れていて、調べるほど正確に見える）。役割に配属して育てる。
- **研究・青写真**：研究投資で技術（青写真）を解放。製品の品質はその青写真が要求する**職業の能力**で決まる（例：Web＝デザイン＋エンジニア／福祉＝セールス）。
- **市場分析**：市場は霧に隠れている。**リサーチャー**で分析すると、規模・競合密度・機会が見える。**空いていて伸びる市場**を先取りするのが狙い。
- **市場は動く**：ヒットで市場は成長し、成功を嗅ぎつけたライバルが参入してくる。殺到する前に地力（品質・信望・営業）で固める。
- **終わり**：資金（ランウェイ）が尽きたらゲームオーバー。勝利条件はなく、自分で目標を決めて遊ぶ箱庭型。

## 開発者向け

TypeScript + Vite。ロジックはブラウザ非依存の純粋関数（`src/core/`）に隔離し、単体テスト（vitest）を備える。

```bash
npm install
npm run dev          # 開発サーバー
npm test             # テスト（134件）
npm run build        # 通常ビルド（dist/）
npm run build:single # 単一HTMLビルド（dist-single/、配布・Pages用）
```

※これは開発中の初期プロトタイプであり、多くのシステムが簡略化・未実装です。
